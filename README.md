## Описание

Данный репозиторий содержит лабораторные работы и практические задания по курсу функционального программирования на языке **Haskell**. Работы охватывают основные концепции функционального программирования от базовых конструкций до продвинутых абстракций.

## Инструментарий

- **Язык:** Haskell
- **Компилятор:** GHC (Glasgow Haskell Compiler)

## Структура курса

Репозиторий организован по темам, каждая из которых представляет отдельный раздел курса:

### Базовые концепции
- **4_introduction** - Введение в Haskell: базовый синтаксис, рекурсия, функции высшего порядка
- **5_1_lists** - Работа со списками: операции, генераторы списков, функции обработки
- **5_2_types** - Типы данных: алгебраические типы данных, системы типов

### Классы типов и их реализация
- **6_standart_classes_realization** - Реализация стандартных классов типов (Eq, Ord, Show, Read и др.)
- **7_semigroups_monoids** - Полугруппы и моноиды: алгебраические структуры в Haskell

### Функторы и аппликативы
- **8_applicative_functors** - Аппликативные функторы: применение функций в контексте
- **9_parsers** - Парсеры: построение комбинаторов парсеров, работа с Traversable и Foldable

### Монады
- **10_monads** - Монады: основы монадических вычислений, do-нотация
- **11_standart_monads** - Стандартные монады: Maybe, Either, List, State, Reader, Writer
- **12_monad_transformers** - Трансформеры монад: композиция монадических эффектов

### Продвинутые темы
- **13_types_computing** - Вычисления на уровне типов: GADTs, type families, phantom types
- **14_recursive_types** - Рекурсивные типы данных: деревья, графы, fix-point комбинаторы
- **15_zippers_lens** - Зипперы и линзы: функциональные ссылки, навигация по структурам данных

### Теория
- **0-3_theory** - Теоретические материалы
- **lambda_calc** - Лямбда-исчисление: основы, редукция, Church encodings
- **lectures** - Материалы лекций

## Основные темы

В рамках курса были изучены и реализованы:

- ✅ **Функциональная парадигма**: pure functions, иммутабельность, композиция функций
- ✅ **Ленивые вычисления**: бесконечные структуры данных, thunks, оптимизация
- ✅ **Система типов**: строгая статическая типизация, вывод типов, полиморфизм
- ✅ **Алгебраические типы**: sum types, product types, pattern matching
- ✅ **Классы типов**: Functor, Applicative, Monad, Foldable, Traversable
- ✅ **Монадические вычисления**: состояние, ошибки, недетерминизм, IO
- ✅ **Парсер-комбинаторы**: построение парсеров через композицию
- ✅ **Продвинутые возможности**: GADTs, рекурсивные схемы, зипперы, линзы
- ✅ **Лямбда-исчисление**: теоретические основы функционального программирования

- Применение функциональных паттернов для решения практических задач
